Comment Creer un projet ?
  • Aller au a la barre de menu choisissez
  • le menu fichier>Nouveau...>projet.


[Haut]
Comment Creer un diagramme de classe ?
  • Aller au a la barre de menu choisissez
  • le menu fichier>Nouveau...>diagramme de classe.


[Haut]
Comment ajouter une classe?
  • Aller au a la barre de menu choisissez
  • le menu fichier>Nouveau...>diagramme de classe.
  • Boite a outils devrait aparaitre a gauche de l'ecran > Elements >classe.


[Haut]
Comment ajouter un attribut a une classe?
  • Clique droit sur la classe > Ajouter un attribut
  • Un panneau de propriété sera afficher en bas a droit pour remplir les champs de l'attribut


[Haut]
Comment creer une methode?
  • Clique droit sur la classe > Ajouter un attribut
  • Un panneau de propriété sera afficher en bas a droit pour remplir les champs de l'attribut


[Haut]
Comment creer une relation ?
  • Clique sur >Relations dans la boite a outils a gauche puis choissiez la relation convenable


[Haut]
Comment modifier un attribut/ou methode ?
  • Cliquez sur la classe avec le boutton droit de la souris > Modifier les attributs/Methodes


[Haut]
Comment ajouter un actor/action?
  • Boite a outils devrait aparaitre a gauche de l'ecran > Elements >Acteur/Action.

[Haut]
Comment creer une diagramme de use case ?
  • Aller au a la barre de menu choisissez
  • le menu fichier>Nouveau...>diagramme de cas d'utilisation.


[Haut]
Comment copier/couper/coller/supprimer un actor ?
  • Boutton droit de la sourie sur l'acteur choisissez > copier/coller/couper/supprimer


[Haut]
Comment creer un package/grouper/degrouper
  • Selection les elements a grouper en glissant la souris sur eux .
  • Appuyez sur le boutton grouper en bas a droit .



[Haut]
Comment Generer un code java ou c++ ?
  • Aller au a la barre de menu choisissez
  • le menu Outils>Generer du code...>Java/C++.
